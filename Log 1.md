## Log 1

### Java语言特点

``` Java语言特点
面向对象性
两个要素：类、对象
三个特征：封装、继承、多态

健壮性
1.去除了C语言中的指针
2.自动的垃圾回收机制 --> 仍然会出现内存溢出，内存泄漏
3.跨平台性：write onece, run anywhere(一次编译，到处运行）————归功于JVM
```

### JDK,JRE,JVM的关系

``` 
JDK=JRE+开发工具集（例如javac编译工具等）
JRE=JVM+JAVA SE标准类库
```

### 环境变量的配置

```
path环境变量：windows操作系统命令时索要搜寻的路径
为什么配置环境变量：希望java开发工具(javac.exe,java.exe)在任何文件路径下都可以执行成功
```
